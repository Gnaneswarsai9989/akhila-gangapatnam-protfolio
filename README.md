## 🚀 How to Use

### Step 1 — Add Your Photo
- Place your profile photo inside the `images/` folder
- Rename it exactly: `akhila.jpg`
- Best size: **600×750px** or taller portrait (3:4 ratio)
- The website automatically handles missing images with a beautiful gradient fallback

### Step 2 — Open the Website
- Simply open `index.html` in any browser
- Or drag & drop `index.html` into Chrome/Firefox

### Step 3 — Deploy Online (Free)
**Option A — Netlify (Recommended):**
1. Go to [netlify.com](https://netlify.com)
2. Drag the entire `akhila-portfolio` folder into the deploy area
3. Your site is live instantly!

**Option B — GitHub Pages:**
1. Create a GitHub account
2. Upload all files to a repository
3. Go to Settings → Pages → Deploy from main branch

## ✏️ Customizing

### Change Colors
In `css/style.css`, edit the `:root` variables:
```css
--accent: #c9a96e;      /* Gold accent color */
--black: #000000;       /* Background */
```

### Change WhatsApp Number
Search for `919346159120` in both `index.html` and `js/main.js` and replace.

### Add Real Work Videos
In `index.html`, replace the `.work-thumb-bg` divs with actual video thumbnails:
```html
<img src="images/project1-thumb.jpg" class="work-thumb-img" />
```
